# Code Quality Check and Fix

このプロジェクトのコード品質を確認し、必要に応じて修正してください。

## 実行するタスク

1. **Ruff Format**
   - `uv run ruff format .` を実行してコードをフォーマット
   - フォーマットされたファイルがある場合は報告

2. **Ruff Check**
   - `uv run ruff check .` を実行してリントチェック
   - エラーや警告がある場合は内容を確認
   - 自動修正可能なものは `uv run ruff check . --fix` で修正
   - 手動修正が必要なものは該当箇所を特定し修正

3. **Mypy Type Check**
   - `uv run mypy .` を実行して型チェック
   - 型エラーがある場合は内容を確認し修正
   - 必要に応じて型ヒントを追加

4. **Pytest**
   - `uv run pytest` を実行してテストを実行
   - テストが失敗した場合は原因を特定
   - テストコードまたは実装コードを修正

## 修正方針

- すべてのエラーと警告を解決すること
- コードの可読性を保ちながら修正
- 型ヒントは明確かつ正確に記述
- テストが通るまで修正を繰り返す
- 修正内容を簡潔に報告

## 最終確認

すべてのチェックとテストが通ったことを確認し、以下を報告してください:
- 修正したファイルと内容
- 最終的なチェック結果（ruff format, ruff check, mypy, pytest）
- コード品質の状態
